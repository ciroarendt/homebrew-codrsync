# Homebrew Tap for codrsync

This is the official Homebrew tap for [codrsync](https://codrsync.dev) - the AI-powered development orchestrator.

## Installation

```bash
# Add the tap
brew tap ciroarendt/codrsync

# Install codrsync
brew install codrsync
```

## Usage

After installation, you can use codrsync:

```bash
# Check version
codrsync --version

# Initialize a new project
codrsync init

# Scan existing project
codrsync scan

# Check project status
codrsync status
```

## Updating

```bash
brew update
brew upgrade codrsync
```

## Uninstalling

```bash
brew uninstall codrsync
brew untap ciroarendt/codrsync
```

## Alternative Installation

You can also install via pip:

```bash
pip install codrsync
```
